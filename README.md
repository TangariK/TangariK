**What's the deal with me?**

* 🧩 **End-to-end product:** from schema and APIs to the UI — not just screens, not just scripts.
* 🧠 **Always shipping & learning:** if it involves AI, automation, or a new stack, I'm already trying it.
* 🏛️ **Architecture with pragmatism:** clean code and patterns when they help; flexible when the domain and deadline demand it.
* 🔐 **Careful with sensitive work:** secrets stay out of Git, auth done properly, private repos when IP requires it.

**Right now:**

* Shipping production systems at **Ether Exchange** (web apps, APIs, admin/DEX, quality)
* **[Breshopping](https://github.com/TangariK/breshopping)** — multi-tenant SaaS for thrift-store inventory
* Technical write-ups in **[portfolio-cases](https://github.com/TangariK/portfolio-cases)** (no proprietary source code)
